#  IndiSafe — AI Tourist Guardian

##  Hackathon Project

IndiSafe is an AI-powered predictive safety system designed to protect tourists in real time.

---
## Live Demo
https://indisafe-ai-tourist-guardian.streamlit.app
-------
##  Features

- Live tracking map
- Crime risk zones
- Real-time safety status
- IndiAI smart assistant
- Multi-language support
- Nearby hospital & police info
- SOS emergency system
- Fake call safety feature
- Companion tracking
- Predictive danger alerts
